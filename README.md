# Hologram Hacks

**Software worth owning.** An engineering studio in Pittsburgh, PA, building privacy-first products and secure custom software for teams everywhere.

### [hologramhacks.com](https://hologramhacks.com)

## Products

- **[Relay](https://relay.hologramhacks.com)**: private, self-hosted, end-to-end-encrypted team communication. Voice, video, and chat without the tracking. ([case study](https://hologramhacks.com/relay-case-study))
- **[ClipStack](https://github.com/HologramHacks/clipstack)**: a tiny clipboard-history popup for Windows. Single file, zero third-party dependencies, written in Rust.
- **[debloatr](https://github.com/HologramHacks/debloatr)**: a Windows disk-reclaim and debloat playbook.
- **[inbox-detox](https://github.com/HologramHacks/inbox-detox)**: take a Gmail inbox to zero and keep it there.
- **[crawl-space](https://github.com/HologramHacks/crawl-space)**: a Claude Code skill that gets a website crawled, indexed, and found.

# Hologram Hacks

**Software worth owning.** An engineering studio in Pittsburgh, PA, building privacy-first products and secure custom software for teams everywhere.

### [hologramhacks.com](https://hologramhacks.com)

## Products

- **[Relay](https://relay.hologramhacks.com)**: private, self-hosted, end-to-end-encrypted team communication. Voice, video, and chat without the tracking. ([case study](https://hologramhacks.com/relay-case-study))
- **[ClipStack](https://github.com/HologramHacks/clipstack)**: a tiny clipboard-history popup for Windows. Single file of Rust, zero third-party crates, raw Win32, DPAPI-encrypted secrets. Ships with CI, cargo audit, and signed build provenance on every release.

### Agent skills

A toolkit for running [Claude Code](https://claude.com/claude-code) as a reliable operator: each one takes a messy recurring chore, encodes the process with guardrails and failure handling, and makes it repeatable.

- **[inbox-detox](https://github.com/HologramHacks/inbox-detox)**: takes a Gmail inbox to zero and keeps it there. Battle-tested on a real 50,000-message backlog.
- **[debloatr](https://github.com/HologramHacks/debloatr)**: Windows disk-reclaim and debloat playbook. Every destructive step is copy-verify-delete guarded.
- **[crawl-space](https://github.com/HologramHacks/crawl-space)**: gets a new site crawled, indexed, and found.
